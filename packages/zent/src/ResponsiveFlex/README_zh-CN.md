## 响应式列表

 *  子元素宽度随着父元素宽度自适应。      
 *  子元素在水平方向有固定的间隔。     
 *  可指定每一行的元素数量。      
 *  可指定子元素的宽高比例，默认为 1:1 。  

### 使用指南

-   通过 `gutter` 来控制间距
-   通过 `rowVolume` 控制个数.


### API

#### ResponsiveFlex

| 参数        | 说明                          | 类型     | 默认值         | 备选值                                |
| --------- | --------------------------- | ------ | ----------- | ---------------------------------- |
| gutter      | 间距                          | number | `'default'` | `'primary'`、`'danger'`、`'success'` |
| rowVolume      | 列表个数                          | number | `'medium'`  | `'large'`、`'small'`                |
